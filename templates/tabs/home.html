Twitter Analysis Application
The Twitter Analysis Application is a tool for analysing twitter data. Using the Twitter APIs, this application can fetch a set of tweets for a given hashtag or user handle, and draw conclusions based on that dataset. All data accessed is public, and conclusions made by the algorithm are not to be taken as certainties, only observations.

Analyse Tweets
Analyse a set of tweets for a given user handle or hashtag. For that set of tweets, determine:
  • Tweet-set Sentiment
  • Tweet-set Political Leaning
  • Tweet-set Emotions
  • Most Popular Tweet
  • Most Used Emojis in Tweet-set
  • Most Used Words in Tweet-set
  • Most Used Hashtags in Tweet-set
  • Most Tagged Users in Tweet-set
This data can be used to see interesting trends or patterns within a hashtag, and to observe interesting data about how public figures use twitter.
Compare Tweets
Analyse two sets of tweets for given user handles or hashtags side by side to see similarities and differences.
  • Tweet-set Sentiment
  • Tweet-set Political Leaning
  • Tweet-set Emotions
  • Most Popular Tweet
  • Most Used Emojis in Tweet-set
  • Most Used Words in Tweet-set
  • Most Used Hashtags in Tweet-set
  • Most Tagged Users in Tweet-set
This data can be used to see how different issues are discussed in different ways on twitter, and determine how different groups or political figures use different words, emotions and sentiments within their tweets
Analyse Account
Analyse a given public twitter user's account and recent activity.
  • Tweet-set Sentiment
  • Tweet-set Political Leaning
  • Tweet-set Emotions
  • Most Popular Tweet
  • Most Used Emojis in Tweet-set
  • Most Used Words in Tweet-set
  • Most Used Hashtags in Tweet-set
  • Most Tagged Users in Tweet-set
As well as:
  • Twitter User Account Info
  • Account Authenticity Probability
  • Bot type Probabilities

Political Analysis Datasets
The dataset used in the Political Leaning prediction can be set based on country, since political discourse varies a lot across different countries. The data used to classify american politicians tends not to be very accurate at classifying Irish or politicians, since general speech, and especially political discourse, is very different in the different countries. The political analysis region selctor, shown below as an example, allows you to choose what the country from which the political database is gathered. The default 'Global' option is a collection of the databases for Ireland, the Uk and the US. This dataset can make general predictions, but will be less accurate on average at predicting the political leaning than the correct country datset for the set of tweets.


Result Fields Explanations
The conclusions which this application draws based on the given twitter data are:
  • Tweet-set Sentiment: How Positive or negative the set of tweets are.
  • Tweet-set Political Leaning: Where this set of tweets falls on the spectrum from left/liberal to right/conservative. The political sphere to base this prediction on can be chose as Ireland, the UK, the US or Global.
  • Tweet-set Emotions: Each tweet is assigned a predominant emotion based on the words used. This is used to show the prevelence of various emotions within the set of tweets.
  • Most Popular Tweet: The Most Liked and retweeted tweet by the user or wihin the hashtag from the last 7 days.
  • Most Used Emojis in Tweet-set: The most frequently used emojis within the set of tweets.
  • Most Used Words in Tweet-set: The most frequently used words within the set of tweets.
  • Most Used Hashtags in Tweet-set: The most frequently used hashtags within the set of tweets.
  • Most Tagged Users in Tweet-set: The most frequently tagged twitter users within the set of tweets.
When analysing a twitter account, this additional data can be seen:
  • Twitter User Account: The user photo, twitter bio, follower and following stats are shown,along with the account age and number of tweets made by the account.
  • Account info (If avaliable): If an account has its location set or a tweet pinned to the top of their twitter timeline, then those fields will also be shown.
  • Account Authenticity Probability: The probability of a twitter account being authentic, and not a bot, is calculated using Botometer based on the last 7 days of tweets.
  • Bot type Probabilities: The probability of the Account being any of 5 different types of common twitter bots.
